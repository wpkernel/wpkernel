# Function: createPhpBaseControllerHelper()

```ts
function createPhpBaseControllerHelper(): BuilderHelper;
```

Creates a PHP builder helper for the base REST controller.

This helper generates the `BaseController.php` file, which serves as the
foundation for all other REST controllers in the generated PHP output.

## Returns

[`BuilderHelper`](../type-aliases/BuilderHelper.md)

A `BuilderHelper` instance for generating the base REST controller.
